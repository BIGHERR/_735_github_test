Программа для работы с побитовым доступом к данным с помощью трех разных способов: «bit-banding», «bit field» и «ручная» работа с битами.

Для запуска программы необходимо создать проект в IAR Embedded Workbench 6.30.1.3142 (в данной среде разрабатывалась программа) и настроить его в соответствии с документацией по настройке IAR с сайта Миландр, заисключением того, что отладчик устанавливаем в режим «симуляции». После в созданный проект добавляем файл «main.c».
